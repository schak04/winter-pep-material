1.Events:-
Events are actions like click, keypress, submit, etc. that happen on a webpage.
They are used to handle user interaction, update content, validate forms, and improve user experience.


Examples of Events

Clicking a button → click
Typing in input → keyup
Moving mouse → mousemove
Submitting form → submit
Page loading → load
Scrolling → scroll

We use events to make websites interactive.
Without events:
❌ Website = Only text and images (no interaction)

With events:-Purpose of Using Events (Why They Are Important)

You use events to:

Handle User Actions

  **Event Bubbling**

  //Event bubbling is when an event starts from target element and moves upward to parent elements.

  //Event bubbling allows events to move from child to parent, which helps in event delegation, reduces listeners, and improves performance.
  //Purpose of event bubbling is to handle multiple child events using one parent listener and improve performance.

  **Event Delegation**

  //Event delegation means:
   Event delegation is a technique where a single event listener is added to a parent element to handle
  events of its child elements using event bubbling.
   document.querySelectorAll("li").forEach((li) => {
    li.addEventListener("click", () => {
      alert(li.innerText);
    });
  });

  let ul = document.querySelector("ul");
ul.addEventListener("click", (e) => {
  if (e.target.tagName === "LI") {
    alert(e.target.innerText);
  }
});

  5. To explain how data can be made dynamic(usually data is being fetched from data base)
     let items = ["apple", "banana", "mango", "orange"];

  function GenerateList() {
    let listContainer = document.getElementById("List");
    listContainer.innerHTML = "";
    items.forEach((ele) => {
      let list = document.createElement("li");
      list.innerText = ele;
      listContainer.appendChild(list);
    });
  } 

  //Keeping data dynamic using input tag using oninput event
   <input
      type="text"
      id="inputTag"
      placeholder="type the text..."
      oninput="updateText(event)"
    />
    <li id="List"></li>

   function updateText(event) {
    console.log(event.target.value);
    document.getElementById("List").innerText = event.target.value;
  }
