Frontend vs Backend:-

//Frontend
The part of a website users can see and interact with.
Built using HTML, CSS, JavaScript, React, etc.
Runs in the browser.
Example: buttons, forms, images, product list UI.

Student analogy:
Frontend = The design and screen of a mobile app.

//Backend
The part that works behind the scenes.
Handles data, logic, authentication, APIs.
Built using Node.js, Express, Java, Python, PHP, etc.
Runs on a server.

Student analogy:
Backend = The brain that processes requests and sends data.
