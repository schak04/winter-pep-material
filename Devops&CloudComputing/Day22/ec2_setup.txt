Ec2:-A virtual server in the cloud.
EC2 allows us to run computers (servers) on AWS without buying physical machines.

Q.Why Do We Need EC2?
Ans:-
Traditionally:--->
You had to buy servers 🖥️
Maintain hardware
Handle scaling manually

With EC2:--->
Launch server in minutes ⚡
Scale up/down easily
Pay only for usage 💰
---------------------------------------------------------------------------------
Ec2 Setup steps:-
(a). Look for EC2 and got to it's dashboard
(b). Click on launch instance 
(c). Now create a tag by giving proper name to it(you can opt for multiple resources from it),Tags are imp while filtering.
(d). To select the instance opt for an Ami given(amazon linux which is similar to centos).
(e). Select amazon ami(free tier available) and save the user(will be used to login).
(f). Now select the instance type:-
    -Instance type tells how powerful your EC2 machine is
    -CPU (vCPU):- Number of virtual CPUs,Determines processing power
    Ex:-
    t2.micro → 1 vCPU
    c5.large → 2+ vCPU
    -Memory (RAM):-Amount of RAM available,Affects how many tasks can run
    Ex:-
    t2.micro → 1 GB RAM
    r5.large → high RAM
    -Storage:-Type of storage supported like -->EBS (most common) & Instance store (temporary)
    -Network Performance
    -IOPS / Throughput:-Disk performance (read/write speed)
(g). Now go to key-pair(login)
    - It has two ip's one public and other private. Give a proper name to it and opt for .pem(will use in gitbash)

(h).Go to network settings edit it 
    - in source look for myip(you ip public ip address)

(i). Look in advance details to paste some important commands in the bottom
    - These are some useful commands which our instance runs when it is launched

    ```
    #!/bin/bash
yum update -y
yum install -y httpd
systemctl start httpd
systemctl enable httpd
echo "<h1>Hello from EC2</h1>" > /var/www/html/index.html

or 

#!/bin/bash
dnf update -y
dnf install -y httpd
systemctl start httpd
systemctl enable httpd
echo "<h1>Welcome to EC2</h1>" > /var/www/html/index.html

Note:- Both commnads will work same as amazon linux is same as centos 

```
